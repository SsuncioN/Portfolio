Stack Tecnológica

Desenvolvi um sistema de controle gestual em tempo real utilizando visão computacional (OpenCV + MediaPipe + Math + AutoGUI). O sistema interpreta movimentos das mãos e os converte em comandos de teclado com baixa latência, criando uma interface natural entre usuário e computador/jogos. O projeto está em fase ativa de aprimoramento, com melhorias voltadas para redução de latência, aumento de robustez e refinamento dos algoritmos de reconhecimento gestual.

Arquitetura

Captura de vídeo → OpenCV

Detecção de mãos → MediaPipe

Cálculo vetorial → Math

Automação de input → AutoGUI

Loop contínuo com controle de estado
